# 進捗状況を表示

`.claude/progress.json` を読んで以下を報告:

1. **全体進捗**: 完了数 / 総数 (110)
2. **ディレクトリ別進捗**: 各ディレクトリの完了状況
3. **次のファイル**: 次に作成するファイル名とトピック
4. **最近の完了**: 直近に完了したファイル

## 出力例

```
=== Zig by Examples 進捗 ===

全体: 4/110 (3.6%)

ディレクトリ別:
  01_basics:     4/15  ████░░░░░░
  02_structs:    0/10  ░░░░░░░░░░
  ...

次: n005_integers.zig (整数型)
```
