# 連続ファイル作成モード

`/next` を**ユーザーが割り込むまで繰り返し実行**する。

## フロー

```
┌─────────────────────────────────────┐
│  1. /next と同じ処理を実行          │
├─────────────────────────────────────┤
│  2. 簡潔に報告                      │
│     ✓ n00X_topic.zig 作成完了       │
│     → 次: n00Y_next_topic.zig       │
├─────────────────────────────────────┤
│  3. 自動的に次へ進む                │
└─────────────────────────────────────┘
```

## 停止条件

- ユーザーが割り込んだ場合
- 全ファイル完了（n110まで）
- 修正不能なエラーが発生した場合
